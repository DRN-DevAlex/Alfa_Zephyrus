## 🚀 Sobre el Proyecto

Esta es la Landing Page oficial del equipo de esports **Alfa Zephyrus**. Diseñada desde cero para reflejar la identidad competitiva del equipo, mostrar a nuestros jugadores (Roster Principal y Academia), próximos eventos y resultados. 

Cuenta con un diseño oscuro (Dark Mode nativo), toques en colores neón metálicos y animaciones fluidas para dar una experiencia de usuario de nivel "clase mundial".

> **🌍 En vivo:** Puedes ver la página funcionando aquí: [👉 alfazephyrus.vercel.app](https://alfazephyrus.vercel.app)

---

## ✨ Características Destacadas

- 🎬 **Splash Screen Inteligente:** Pantalla de carga animada que saluda al usuario con el logo del equipo. Utiliza `sessionStorage` para mostrarse solo una vez por sesión y no interrumpir la navegación.
- 🌌 **Animaciones de Scroll (Reveal):** Los elementos aparecen suavemente de abajo hacia arriba conforme el usuario va bajando por la página.
- 📱 **Diseño 100% Responsivo:** Adaptado a monitores ultra anchos, laptops, tablets y dispositivos móviles.
- 🕹️ **Carrusel de Eventos Interactivo:** Slider horizontal con botones de control para ver las próximas competiciones del equipo.
- 🎨 **Estilo Neón Personalizado:** Uso de variables CSS para mantener una paleta de colores consistente (`--bg-color`, `--accent-metallic`).
